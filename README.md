# OPS Memory Landing

Landing page autonome de la mémoire IA OPS.

Le dépôt contient uniquement le site public, ses médias et les compositions HyperFrames desktop et mobile. Il ne dépend pas du dashboard OPS ni de l'infrastructure de mémoire centrale.

## Prévisualisation locale

```sh
npx serve .
```

## Déploiement

Le projet est déployable directement comme site statique sur Vercel.
